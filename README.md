# JPaint

## Version 1.1 - 09/23/2022

### Features:

-  Open File
  
-  Save File
  
-  Save File As
  
-  Add Colored Border to Image

-  Clear Canvas

-  Draw Lines of Specified Width and Color

-  Scroll Image

-  Many different shape tools

-  Canvas resize
